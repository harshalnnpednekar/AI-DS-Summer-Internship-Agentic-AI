# AI-DS-Summer-Internship-Agentic-AI

A unified platform for automated attendance tracking, academic calendar management, and defaulter list generation — built for the AI & DS faculty.

## Project Structure

- `frontend/` - React-based user interface for the Department Management Agent.
- `backend/` - Server-side application and APIs.
- `AcademicCalendarExtractor/` - Automated parsing and extraction tool for Academic Calendars.

## Setup Instructions

### Frontend Setup
To run the frontend dashboard:

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open the application in your browser at `http://localhost:5173`.

*For more details on the frontend application, please refer to [frontend/README.md](frontend/README.md).*
